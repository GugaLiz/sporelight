## 第一部分：动画分镜表

以下按时间顺序列出 Boss 关所有动画/特效的详细分镜描述。

---

### 分镜 01：进入关卡 - 环境铺垫

| 项目 | 内容 |
| :--- | :--- |
| 时长 | 3秒 |
| 触发条件 | 关卡加载完成 |
| 画面元素 | 洞穴背景 + 中央水晶花（被缠绕，闭合） + 4个灰色苔藓（四角） + 漂浮孢子 |
| 动画/特效 | 水晶花有呼吸浮动（上下 5px，周期 2秒）。漂浮孢子缓慢飘动。 |
| 音效 | 洞穴环境音（水滴声+风铃声） |

---

| 项目 | 内容 |

### 分镜 03：孢子收集

| 项目 | 内容 |
| :--- | :--- |
| 时长 | 1秒（单个） |
| 触发条件 | 露薇触碰孢子 |

| 音效 | “叮”声（清脆，音调渐高） |

### 分镜 04：顺序提示（花瓣光点闪烁）

| 项目 | 内容 |
| :--- | :--- |
| 时长 | 循环，每轮约3.2秒 |
| 触发条件 | 进入Boss战后持续播放 |
| 画面元素 | 水晶花周围的4个花瓣状光点 |
| 动画/特效 | 光点按正确顺序依次闪烁：亮起（0.3秒）→ 熄灭（0.2秒）→ 间隔（0.3秒）→ 下一个亮起。每个光点闪烁时伴有轻微放大（120%）。 |
| 音效 | 每个光点亮起时有轻柔的“滴”声 |
