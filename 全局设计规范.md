【游戏名称】《微光苔原》（英文名：Sporelight）
【主角名称】露薇儿（Luvie）
【游戏类型】治愈系解谜冒险网页游戏（HTML5 / Codex 开发）

【风格关键词】
- 纪念碑谷风格（低多边形、柔和阴影、几何构成）
- 莫兰迪色系 + 温暖的微光点缀
- 圆润、可爱、无棱角
- 画面有“呼吸感”（元素轻微浮动、光晕）

【画布/页面尺寸规范】
- 游戏主画布尺寸：1280×720 像素（16:9 宽屏）
- UI 安全区：左右各留 40px，上下各留 30px
- 分辨率适应：优先 1280×720，向下兼容 1024×576（等比缩放）

【切图与素材要求】
- 所有角色、物件、UI 元素均输出为 PNG 格式，透明背景
- 角色/动画帧尺寸：建议 150×150px 以内（具体根据动作幅度可略大）
- 世界元素（花台、苔藓等）：尺寸 80×80px ～ 120×120px
- UI 按钮：最小 60×60px（触控友好）
- 输出时请按以下命名规则：类型_名称_状态.png
  例如：char_luvie_idle.png、ui_btn_sow.png、world_moss_glow.png
