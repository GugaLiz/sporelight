# 微光苔原 / Sporelight 命名页切图包

本包根据“命名页面输出清单.md”制作，包含命名页背景、Logo、标题、输入框、按钮三状态、提示文字、露薇命名页立绘、特效与装饰。

## 目录
- backgrounds/：命名页背景与遮罩
- ui/logo/：命名页 Logo、英文副标题、标题文字
- ui/input/：输入框三状态与光标
- ui/buttons/：随机名字、恢复默认、开始旅程按钮状态
- ui/text/：提示文字与按钮标签
- characters/：露薇命名页立绘与投影
- fx/：孢子、输入框光晕、按钮闪光
- deco/：角花装饰
- meta/asset_manifest.json：尺寸与九宫格说明

## 建议
输入框与按钮可使用 PNG 九宫格拉伸，也可以直接用 CSS 重建，PNG 作为视觉参考。
